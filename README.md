# D.D(mood weather)

-앱설명
사용자의 하루 일어났던 일을 감정을 포함해 일기로 작성하고 감정들을 날씨로 카테고리를 만들어 편하게 다시 볼수있는 앱입니다.

------------
UIkit의 sotryboard를 사용하고 swift언어를 사용함
